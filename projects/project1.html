RAW TYPOGRAPHY

PROJECT SPECS

  • DATE: 2023
  • ROLE: Typography Design
  • TOOLS: HTML, CSS
  • CLIENT: Self-initiated

This project explores the raw essence of typography in brutalist web design. By stripping away decorative elements and focusing on pure typographic hierarchy, this exercise demonstrates how brutalist principles can be applied to create impactful digital experiences.

The design intentionally exposes the structure of the typographic system, embracing monospace fonts, stark contrast, and unfiltered presentation of content. Every element serves a specific function with no unnecessary decoration.

BODY TEXT

This is standard body text in a monospace font. Notice the raw, unfiltered presentation and deliberate spacing. Typography in brutalism is about honesty in presentation.

Line height is carefully considered for readability while maintaining the raw aesthetic of brutalist design.

CODE BLOCK
body { font-family: monospace; line-height: 1.4; } h1, h2, h3 { text-transform: uppercase; font-weight: bold; }
