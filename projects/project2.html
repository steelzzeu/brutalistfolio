EXPOSED GRID

PROJECT SPECS

  • DATE: 2023
  • ROLE: Grid System Design
  • TOOLS: HTML, CSS Grid
  • CLIENT: Self-initiated

This project demonstrates the raw power of grid systems in brutalist design. Unlike traditional web designs that hide underlying structure, this approach deliberately exposes the grid to create visual tension and highlight the construction of the page.

By making the grid visible and integral to the design, we emphasize the digital medium's native qualities rather than trying to mimic physical materials. The result is an honest presentation of web design that embraces its inherent structure.

RAW CSS GRID CODE

The grid layout above is created using CSS Grid with deliberately exposed coordinates and structure:

.base-grid {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  grid-template-rows: repeat(6, 100px);
  gap: 10px;
}

.item-1 {
  grid-column: 1 / 5;
  grid-row: 1 / 3;
}
