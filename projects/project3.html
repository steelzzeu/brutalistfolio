RAW FORMS

PROJECT SPECS

  • DATE: 2023
  • ROLE: UI Design
  • TOOLS: HTML, CSS, JavaScript
  • CLIENT: Self-initiated

This project explores the brutalist approach to form design. Instead of hiding the native form elements behind ornate styling, this approach celebrates the raw functionality of web forms while maintaining a distinctive visual identity.

By stripping away decorative elements and focusing on the structural aspects of form controls, we create an interface that is both utilitarian and visually striking. The result is a form that speaks honestly about its digital nature.

RAW FORM DESIGN PRINCIPLES

The brutalist form above follows these key principles:

  • Intentionally visible structure
  • Custom styling that emphasizes rawness
  • Monospace fonts for all text elements
  • High contrast black-and-white palette
  • Deliberate exposure of form architecture

.raw-form textarea {
  width: 100%;
  background-color: transparent;
  border: none;
  border-bottom: 3px solid var(--black);
  padding: 0.5rem 0;
  font-family: monospace;
}
